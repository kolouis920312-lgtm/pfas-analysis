# pfas_toolkit/data — 內建參考資料

## ml_phi_reference.csv — PFAS 氣粒分配 φ（ML 參考基準）

本專案**機器學習模型**對 PFAS 氣粒分配的輸出，供「氣粒分配」方法的 **ML 參考** 模型按
化合物名比對使用，並作為其他文獻模型的**驗證基準**。

- **來源**：`PFAS氣粒分配_彙整表.xlsx`（階層貝氏＋設限感知模型；以真實 LOD/LOQ 設限的實測資料校驗）。
- **內容**：38 支 PFAS，8 族（PFCA/PFSA/FTS/FOSAA/PFECA-ether〔離子〕、FTOH/FOSA/FOSE〔中性〕）。
- **機制感知**：離子型走碳數預測、中性走 log P_L° 預測。

| 欄位 | 說明 |
|---|---|
| `compound` | 化合物名（比對鍵，不分大小寫） |
| `family` | 族別 |
| `mechanism` | 離子／中性 |
| `Cn` | 全碳數 |
| `phi_25C` | 25°C 粒相佔比 φ=Cp/(Cp+Cg)（後驗均值；0→全氣相、1→全粒相） |
| `phi_lo89` / `phi_hi89` | 89% 可信區間下/上界 |
| `phase` | 相態判讀（氣相主導<0.2／偏氣相0.2–0.5／偏粒相0.5–0.8／粒相主導≥0.8） |

**注意**：φ 為固定條件（PM10 代理 TSP、25°C）的後驗值，**不隨 TSP 即時調整**，定位為驗證基準
（要做情境/外推請用公式類模型）。**Yamazaki 鏈長式對長鏈 PFCA 會低估粒相，PFCA 請以本表為準。**
